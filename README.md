## ✨ 功能特性

### 🔌 Session Management（会话管理）
- 实时查看 MySQL 活跃会话
- 支持筛选执行中 / 空闲会话
- 一键 Kill 慢 SQL / 异常会话
- 支持批量操作与自动刷新

### 📊 Disk Usage Analysis（磁盘使用分析）
- 实例级 / 数据库级 / 表级磁盘占用统计
- InnoDB 表空间大小分析
- 支持排序、搜索与可视化展示
- 辅助排查磁盘暴涨问题

### 💻 SQL Execution（SQL 执行）
- 多连接管理
- SQL 编辑与执行
- 执行结果表格化展示
- 支持大结果集分页显示
- SQL 执行耗时统计
